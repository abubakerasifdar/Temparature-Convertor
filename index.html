Temparature Convertor

Write a program that get the number and a letter. if the letter is "f", the program should treat the number enterd as temperature in degree fahrenheit and convert it to the temperature in degree Celsius and print a suitable message. If the letter is "c", the program should consider the number as Celsius tempareture and convert it to Fahrenheit temperature and print a suitable message. The program should display error message and the exit if the use enters any other letter.
